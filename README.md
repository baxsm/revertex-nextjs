## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/baxsm/revertex-nextjs.git
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) in your browser.
